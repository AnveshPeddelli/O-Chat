# Resume Bullets

- Built a local-network office chat prototype in C++ using standalone ASIO, TCP sockets, and CMake.
- Implemented host/join modes, multi-client session management, and asynchronous message broadcasting.
- Improved console UX with structured prompts, connection status messages, and readable chat formatting.
- Debugged runtime networking issues including ASIO event-loop ordering and signed integer port overflow.
- Designed the app around ephemeral communication with no persistent message history or database storage.
- Began integrating shared-key encrypted message relay to strengthen privacy for LAN communication.
